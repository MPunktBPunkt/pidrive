📡 PPM Frequenzkorrektur
PPM
…
Was ist PPM? Quarzfehler des RTL-SDR Sticks (typisch ±20–100 ppm).
Bei 100 MHz: 50 ppm = 5 kHz Versatz → schlechtere FM-Stereo-Trennung, DAB-Aussetzer.
Wie kalibrieren? 🔬 Kalibrieren drücken (30s, Stick darf nichts anderes tun) → Wert übernehmen → beim nächsten FM/DAB Start automatisch aktiv.
Oder manuell: FM-Sender hören → ±10 ppm ausprobieren → besser klingt = richtig.
0 = deaktiviert (Werkseinstellung). Nach Übernehmen neu abspielen.
